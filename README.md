#Vision-Guided Cartesian Gantry Robot for Object Detection and Positioning

A vision-guided Cartesian robotic system that combines YOLOv8-based object detection, ESP32 motion control, and a custom-built two-axis gantry mechanism to automatically detect objects and position the end-effector at the detected target.

The system integrates computer vision, embedded systems, and precision motion control to create a low-cost robotic positioning platform suitable for industrial automation, academic research, and vision-guided robotics.
